## Tech Stack

**Client:** React, Redux, TailwindCSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/Nurit-Ltd/NurAdmin-Docs.git
```

Go to the project directory

```bash
  cd NurAdmin-Docs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
